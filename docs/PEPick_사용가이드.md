## 홈 화면에 추가하기 (앱처럼 사용)

설치 없이 홈 화면에 아이콘을 추가하면 앱처럼 바로 실행할 수 있습니다.

### iPhone / iPad — Safari

1. **Safari**로 https://pepick.netlify.app 접속
2. 하단 중앙의 **공유 버튼** (네모에서 화살표 나오는 아이콘) 터치
3. 아래로 스크롤 → **"홈 화면에 추가"** 터치
4. 이름 확인 후 우측 상단 **"추가"** 터치

### iPhone / iPad — Chrome

1. **Chrome**으로 https://pepick.netlify.app 접속
2. 상단의 **공유 버튼** (네모에서 화살표 나오는 아이콘) 터치
3. **"더보기"** 터치
4. **"홈 화면에 추가"** 터치
5. 이름 확인 후 **"추가"** 터치

### Android (Chrome)

1. **Chrome**으로 https://pepick.netlify.app 접속
2. 우측 상단 **점 세 개 메뉴 (⋮)** 터치
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 터치
4. 이름 확인 후 **"추가"** 터치
5. 홈 화면에 PEPick 아이콘이 생성됩니다

> Android에서는 접속 시 하단에 "앱 설치" 배너가 자동으로 뜰 수도 있습니다.
> 배너가 뜨면 바로 터치하면 됩니다!

- 브라우저 주소창 없이 **전체 화면**으로 실행

## 시작하기 (최초 1회, 약 3분)

### 1단계: 로그인

- https://pepick.netlify.app 접속
- **구글 계정으로 로그인** (학교 구글 계정 또는 개인 계정)
- 로그인하면 학급 데이터가 클라우드에 자동 저장되어, 다른 기기에서도 동일하게 사용 가능

### 2단계: 학급 설정 위저드

처음 로그인하면 학급 설정 마법사가 자동으로 시작됩니다.

1. **학교급 선택**: 초등학교 / 중학교 / 고등학교
2. **담당 학년 선택**: 복수 선택 가능 (예: 3학년, 4학년)
3. **학년별 반 수 설정**: +/- 버튼으로 조절
4. **학급별 학생 수 조정**: 기본 20명, 학급별 실제 인원에 맞게 조정
5. **선생님 이름 입력**: 앱에 표시될 이름 (선택사항, 건너뛰기 가능)

> 위저드를 완료하면 모든 학급이 자동 생성됩니다!

### 3단계: 학급 선택

- 학급 목록에서 수업할 학급을 탭하면 바로 술래뽑기 화면으로 이동

## 기능별 상세 사용법

### 1. 술래뽑기

수업 중 술래, 천사, 시범자 등을 랜덤으로 뽑을 때 사용합니다.

**설정 방법:**

1. 상단 탭에서 **술래뽑기** 선택
2. **뽑기 설정**:
   - **술래 수**: 뽑을 술래 인원 (1~10명)
   - **천사 수**: 술래를 도와주는 천사 인원 (0~10명)
   - **분/초**: 게임 타이머 시간 설정
3. **뽑을 학생 설정**:
   - **학급 불러오기**: 미리 저장한 학급의 학생 명단을 한번에 불러옵니다 (가장 편한 방법)
   - **번호순 생성**: 1번~N번으로 빠르게 생성
   - **성별 구분**: 남녀를 구분하여 생성 (남녀 따로 뽑기 가능)
4. 학생 카드를 탭하면 **제외/포함** 토글 (결석생, 부상자 제외)
5. **술래 뽑기!** 버튼 터치

**결과 화면:**

- 술래와 천사가 카드로 표시됨
- **게임 시작** → 전체화면 타이머 시작 (종료 시 알림음)
- **다시뽑기** → 동일 설정으로 다시 랜덤 뽑기
- **중복 제외** 토글 ON → 이전에 뽑혔던 학생은 자동 제외 (공평하게 돌아가며 뽑기)

### 2. 모둠뽑기

학생들을 여러 모둠(팀)으로 나눌 때 사용합니다.

**설정 방법:**

1. 상단 탭에서 **모둠뽑기** 선택
2. **뽑기 모드 선택**:
   - **랜덤**: 매번 새롭게 랜덤 배정
   - **고정**: 학급관리에서 미리 편성한 고정 모둠 사용
3. **뽑기 설정**:
   - **모둠당 인원**: 한 모둠에 몇 명? (2~10명)
   - **모둠 개수**: 몇 개 모둠? (1~12개)
   - **분/초**: 활동 타이머 시간
4. **모둠 이름 설정**:
   - **숫자순**: 1모둠, 2모둠, 3모둠...
   - **학급 설정 이름**: 학급관리에서 미리 정한 이름 사용
   - **즉석 커스텀**: 직접 입력 (예: 독수리팀, 호랑이팀...)
5. **뽑을 학생 설정**: 술래뽑기와 동일 (학급 불러오기 / 번호순 / 성별 구분)
6. **모둠 뽑기!** 버튼 터치

**결과 화면:**

- 각 모둠별로 배정된 학생 카드 표시
- 남는 학생/부족한 학생이 있으면 자동 안내 팝업
- **타이머** → 활동 시간 타이머 (전체화면 가능)
- **다시뽑기** → 같은 설정으로 재배정

### 3. 만능 휘슬

화면 우측 하단의 📣 플로팅 버튼을 누르면 휘슬 패널이 열립니다.

**3가지 모드:**
| 모드 | 소리 | 활용 |
|------|------|------|
| **꾹 누르기** | 누르는 동안 계속 소리 | 짧은 신호, 주의 집중 |
| **길게 삐———** | 한번 탭으로 긴 소리 | 게임 시작/종료 신호 |
| **삐삐삐!** | 3연속 짧은 소리 | 긴급 정지, 집합 신호 |

- 볼륨 슬라이더로 소리 크기 조절 가능
- 타이머 전체화면에서도 휘슬 버튼 사용 가능

### 4. 타이머

술래뽑기와 모둠뽑기 모두에 타이머가 내장되어 있습니다.

- **설정 화면**에서 분/초를 미리 지정
- **결과 화면**에서 게임 시작 또는 타이머 버튼으로 실행
- **전체화면 모드**: 큰 숫자로 표시 → 학생들도 멀리서 볼 수 있음
- **일시정지/재개** 가능
- 종료 시 알림음 자동 재생
- 모둠뽑기에서는 **프리셋 버튼** (30초, 1분, 3분, 5분, 10분) 제공

### 5. 학급 관리

하단 탭의 **학급관리**에서 현재 학급의 상세 정보를 관리합니다.

- **학생 목록**: 편집 버튼으로 학생 추가/삭제/수정
- **모둠 설정**: 고정 모둠 편성 (모둠뽑기 "고정" 모드에서 사용)
- **표시 이름**: 선생님 이름 변경
- **데이터 초기화**: 전체 데이터 삭제 (신학기 리셋용)

**학급 전환:**

- 우측 상단 프로필 사진 클릭 → **학급 선택** → 다른 학급으로 이동
- 매 차시 학급만 바꾸면 세팅 그대로 사용 가능

## 실제 수업 시나리오

### 시나리오 1: 술래잡기 (초등 3~4학년)

> **상황**: 운동장에서 술래잡기. 학생 25명, 술래 2명, 천사 1명.

1. PEPick 앱 열기 → 해당 학급 선택
2. **술래뽑기** 탭 → 술래 수 2, 천사 수 1, 타이머 3분 00초
3. **학급 불러오기**로 학생 명단 로드 → 결석생 카드 탭해서 제외
4. **술래 뽑기!** 터치 → 결과 확인
5. "오늘 술래는 OOO, OOO! 천사는 OOO!"
6. **게임 시작** → 전체화면 타이머 작동
7. 3분 후 알림음 → 라운드 종료
8. 📣 휘슬로 집합 신호
9. **다시뽑기**로 2라운드 시작 (중복 제외 ON이면 이전 술래는 자동 제외)

### 시나리오 2: 모둠 축구 / 피구 (초등 5~6학년)

> **상황**: 체육관에서 모둠 피구. 학생 28명, 4팀으로 나누기.

1. PEPick 앱 열기 → 해당 학급 선택
2. **모둠뽑기** 탭 → 모둠당 인원 7, 모둠 개수 4
3. 모둠 이름 **즉석 커스텀** → "파랑팀, 빨강팀, 노랑팀, 초록팀" 입력
4. **학급 불러오기** → 결석생 제외
5. **모둠 뽑기!** → 결과 확인
6. 학생들에게 결과 화면 보여주기 (큰 글자로 한눈에 확인)
7. **타이머** → 5분 프리셋 선택 → 전체화면으로 경기 시작
8. 시간 종료 → 📣 삐삐삐로 경기 종료 알림

### 시나리오 3: 협동 줄넘기 릴레이 (중학교)

> **상황**: 6개 모둠이 릴레이 줄넘기. 고정 모둠 사용.

1. **학급관리**에서 미리 6개 모둠 편성해 두기 (한번만 하면 됨)
2. **모둠뽑기** → **고정** 모드 선택 → 모둠 순서 섞기 ON
3. **모둠 뽑기!** → 순서가 섞인 고정 모둠 표시
4. "1번째 도전: OO모둠! 준비~"
5. **타이머** 1분 → 전체화면 → 시작
6. 📣 휘슬로 신호 → 다음 모둠 도전

### 시나리오 4: 준비 운동 시간 제한

> **상황**: 수업 시작, 5분 준비 운동 시간.

1. 아무 화면에서 📣 플로팅 버튼 터치
2. 휘슬 패널에서 **길게 삐———** 선택 → 터치 → 준비 운동 시작 신호
3. 모둠뽑기 타이머에서 **5분 프리셋** → 전체화면 → 시작
4. 5분 후 알림 → 📣 삐삐삐로 집합

## 자주 묻는 질문 (FAQ)

**Q. 데이터가 날아가지 않나요?**

> 구글 로그인 상태에서는 클라우드(Firestore)에 자동 저장됩니다. 핸드폰을 바꿔도 같은 구글 계정으로 로그인하면 데이터가 그대로 있어요.

**Q. 인터넷이 안 되는 운동장에서도 쓸 수 있나요?**

> 한번 접속한 뒤에는 오프라인에서도 기본 기능(뽑기, 타이머, 휘슬)이 동작합니다. 다만 학급 데이터 동기화는 온라인일 때만 됩니다.

**Q. 학생 명단을 매번 입력해야 하나요?**

> 아닙니다! 처음에 위저드로 학급을 세팅하면, 이후에는 **학급 불러오기** 한번이면 끝입니다. 학급관리에서 학생 이름도 수정 가능해요.

**Q. 다른 선생님과 데이터가 공유되나요?**

> 아닙니다. 각자 구글 계정에 독립적으로 저장되어 다른 사람의 데이터와 완전히 분리됩니다.

**Q. 앱 설치가 필요한가요?**

> 설치 없이 크롬 브라우저로 접속하면 됩니다. 자주 쓴다면 "홈 화면에 추가"로 앱처럼 아이콘을 만들 수도 있어요.
